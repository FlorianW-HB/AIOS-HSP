# HSP RECHT Agenten

Dieser Bereich enthaelt alle rechtlichen Agenten und Vorlagen, die ausschliesslich fuer HSP RECHT gelten.

## Startbausteine
- Notariat
- Immobilienrecht
- Erbrecht
- Gesellschaftsrecht
- Mandantenkommunikation
- Fristen und Wiedervorlagen

## Regel
Alles, was notarielle oder anwaltliche Wirkung nach aussen entfalten kann, wird vor Versand kontrolliert und freigegeben.
