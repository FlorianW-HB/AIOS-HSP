# HSP STEUER Agenten

Dieser Bereich enthaelt alle steuerlichen Agenten und Vorlagen, die ausschliesslich fuer HSP STEUER gelten.

## Startbausteine
- Steuerberatung
- Gestaltungsberatung
- Jahresabschlussnahe Themen
- Steuerstrafrecht
- Mandantenkommunikation
- Fristen und Wiedervorlagen

## Regel
Steuerliche Inhalte bleiben von rechtlichen HSP-RECHT-Inhalten getrennt, ausser es gibt einen bewusst gemeinsamen Baustein in `aios/shared`.
