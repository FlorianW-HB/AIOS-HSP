# Prompt-Bibliothek

Hier liegen wiederverwendbare Prompts, die fuer beide Gesellschaften oder fuer beide Arbeitsbereiche geeignet sind.

## Struktur
- `intake/` fuer erste Sachverhaltsaufnahme
- `drafting/` fuer Entwuerfe
- `review/` fuer Pruef- und Kontrollprompts
- `client-facing/` fuer Mandantenkommunikation
- `internal/` fuer interne Steuerung und Organisation

## Regel
Prompts duerfen nur so allgemein sein, dass sie ohne Anpassung in beiden Gesellschaften funktionieren.
Spezialfaelle gehoeren in den jeweiligen Bereich unter `aios/hsp-recht` oder `aios/hsp-steuer`.
