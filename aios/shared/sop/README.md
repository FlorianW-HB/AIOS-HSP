# SOPs

Hier liegen die standardisierten Arbeitsablaeufe.

## Inhalte
- Mandatsannahme
- Erstpruefung
- Entwurfsprozess
- Vier-Augen-Prinzip
- Freigabe und Versand
- Nachverfolgung

## Ziel
Jeder wiederkehrende Vorgang soll als klare Abfolge dokumentiert werden, damit er spaeter von weiteren Agenten oder Teammitgliedern reproduziert werden kann.
